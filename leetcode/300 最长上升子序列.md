## 类型

`动态规划` `数组`

## 思路及代码

### 个人思路

动态规划

```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        # 思路：动态规划
        if len(nums) <= 1:
            return len(nums)

        dp = [1 for _ in range(len(nums))]
        for i in range(1, len(nums)):
            for j in range(i):
                if nums[i] > nums[j]:
                    dp[i] = max(dp[i], dp[j] + 1)
        
        return max(dp)
```

### 二分法
另外还可以用二分法。把数组尽量插入一个排序的空表里，每次替换掉比当前值稍大的元素。
