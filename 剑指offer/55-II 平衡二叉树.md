## 类型

`二叉树` `递归`

## 思路及代码

### 个人思路

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def isBalanced(self, root: TreeNode) -> bool:
        self.result = True
        def recursive(node):
            if not node:
                return 0

            left_depth = recursive(node.left)
            right_depth = recursive(node.right)

            if abs(left_depth - right_depth) > 1:
                self.result = False
            return max(left_depth, right_depth) + 1

        recursive(root)

        return self.result

```
