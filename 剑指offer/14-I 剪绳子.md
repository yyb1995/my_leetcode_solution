## 思路及代码

### 个人思路

```python
class Solution:
    def integerBreak(self, n: int) -> int:
        # 思路：贪心
        # 把n尽量拆分成3的和。如果最后余1，那么把最后一个3和1合并成4并拆成2 * 2；如果最后余2，那么最后乘2即可
        if n <= 3:
            return n - 1
        
        if n % 3 == 0:
            return 3 ** (n // 3)
        
        elif n % 3 == 1:
            return 3 ** ((n - 1) // 3 - 1) * 4
        
        else:
            return 3 ** (n // 3) * 2
```
